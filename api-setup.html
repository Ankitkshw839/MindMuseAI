API Setup

To use MindMuseAI, you need to set up your OpenRouter API key. This will allow our app to connect to AI models to provide you with helpful responses.

How to get an API key:

  1. Visit OpenRouter.ai and create an account
  2. Go to the API Keys section
  3. Create a new key with a name like "MindMuseAI"
  4. Copy the API key and paste it below
